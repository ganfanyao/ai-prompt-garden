# 学习指南 📚

## 提示词

```
我是一个{初学者/有点基础}，想学习{主题}。
请用最通俗的语言和类比来解释核心概念。
每讲完一个概念，给我一个小练习。
```

## 使用场景

- 学习新编程语言
- 理解技术概念
- 准备考试
- 了解陌生领域

## 示例

```
用户：我是一个完全不懂的人，想理解什么是 API。
AI：想象你去餐厅吃饭。
你是"用户"，厨房是"后端系统"。
但你不能直接冲进厨房对厨师说"给我做饭"。
你需要通过"服务员"来沟通——这就是 API。
你点菜（发送请求），服务员告诉厨房（处理请求），然后给你端上菜（返回结果）。
简单吗？现在你试试这个练习：用这个类比，解释什么是 API 密钥。
```
